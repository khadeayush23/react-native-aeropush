# react-native-aeropush

A powerful Over-the-Air (OTA) update service for React Native apps that lets you deploy changes instantly—without going through app store reviews. Deliver bug fixes, UI improvements, and new features in real-time while your users stay seamlessly up to date.

## Installation


```sh
npm install react-native-aeropush
```


## Usage


```js
import { multiply } from 'react-native-aeropush';

// ...

const result = multiply(3, 7);
```
